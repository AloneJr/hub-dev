# 🌐 Tradutor Web

Aplicação web de tradução de textos desenvolvida com HTML, CSS e JavaScript, com suporte a múltiplos idiomas, reconhecimento de voz e alternância entre tema claro e escuro.

Este projeto foi criado com foco em estudo, prática de consumo de APIs e construção de portfólio.

---

## 🚀 Funcionalidades

- Tradução de textos em tempo real  
- Suporte a múltiplos idiomas  
- Reconhecimento de voz via microfone  
- Tema claro e escuro  
- Interface simples e intuitiva  

---

## 🛠 Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript (ES6+)  
- API de Tradução (MyMemory / LibreTranslate)  
- Web Speech API  

---

🎯 Objetivo do Projeto

Este projeto tem como objetivo praticar:
- Manipulação do DOM
- Consumo de APIs externas
- Programação assíncrona (async/await)
- Boas práticas de organização de código
- Criação de interfaces interativas
